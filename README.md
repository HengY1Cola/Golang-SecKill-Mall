## 快速开始

> 当然这个只是个学习库，谈不上进行使用
>
> 但是我还是简单写一下如何快速布局，别进去全是红的找不到文件就行了😊

- SQL文件直接到`Navicat`**创建自己的库**然后**创建查询**然后**执行**就好了

```bash
$ export GO111MODULE=on && export GOPROXY=https://goproxy.cn # 开启mod模块以及换源
$ git clone https://github.com/HengY1Sky/GoSecKillMall
$ cd GoSecKillMall
$ go mod tidy
# 进入Goland开始操作吧
```
